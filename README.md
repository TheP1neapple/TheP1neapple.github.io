# Проект: Место
---

## Описание
Учебный проект, позволяющий, после добавления в него js, редактироватьт профиль с помощью попапа и оставлять отметку 'нравится' на понравившихся картинках

### Технологии
Адаптивная верстка, БЭМ, формы, флекс и грид верстка.
